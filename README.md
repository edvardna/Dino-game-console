# 🦖 Dino Game Console

A portable embedded gaming system running a Dino-style game with real-time input, display control, and a rechargeable battery-powered design.

---

## 🚀 Overview
The Dino Game Console is a hardware-based implementation of a simple endless runner game, inspired by the Chrome Dino game.

The system uses a microcontroller to handle gameplay logic, process button inputs, and render graphics on a small display.  
It is fully portable, powered by a rechargeable battery system, making it a complete standalone embedded device.

---

## 🧠 Features
- Real-time game loop implementation  
- Physical button input (jump / control)  
- Smooth animation on embedded display  
- Score tracking system  
- Fully portable battery-powered operation  

---

## 🛠 Technologies Used
- **Microcontroller (Arduino / embedded platform)**  
- **C / C++** for game logic  
- **OLED display** for graphics output  
- **Digital input buttons** for user interaction  
- **18650 Li-ion battery system**  
- **Battery charging module TP4056**  
- **Voltage regulation circuitry**  

---

## 🔋 Power System
The console is powered by a rechargeable **18650 lithium-ion battery**, allowing fully portable operation.

The power system includes:
- Battery charging module for safe charging  
- Voltage regulation for stable system operation  
- Efficient power delivery to the microcontroller and display  

This demonstrates practical implementation of **battery-powered embedded system design**.

---

## ⚙️ System Architecture
- **Microcontroller**
  - Handles game logic (movement, collisions, scoring)
  - Processes input from buttons
  - Updates display frames

- **Display**
  - Renders player, obstacles, and score

- **Input**
  - Button triggers jump action

- **Power System**
  - Provides regulated power from a rechargeable battery
